CSS Help

Flex Items

align-self

Overrides the align-items property for an individual flex item.

.i3 { align-self: flex-end; }

experiment with flex-start, stretch and center

order

Flexbox orders all the elements according to their order number, in ascending order. The default value for all the elements is 0.

Elements with a lower order number are shown in the beginning.

.i3 { order: -1; }

flex-grow

Gives an item the ability of an element to grow.

Specify an integer.

.i3 { flex-grow: 1; }

flex-basis

Used to set the width of a flex item.

.i3 { flex-basis: 400px; }

flex-shrink

Allows an element to shrink.

flex shorthand.

flex: <grow> <shrink> <basis>;
